# Sistema de Conta Bancária em Java

Projeto desenvolvido em Java com o objetivo de simular operações básicas de um sistema bancário.

## Funcionalidades
- Depósito
- Saque
- Extrato
- Saldo

## Tecnologias
- Java
- IntelliJ IDEA

## Conceitos aplicados
- Orientação a Objetos
- Regras de negócio
- Separação de camadas
